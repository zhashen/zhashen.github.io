Notes on Regular Languages (part1) - Overview

Overview

Let $\Sigma$ be a finite set of symbols and called an alphabet. A (possibly empty) string $w \in \Sigma^*$ is simply an ordered subset of the alphabet. We will often use $\epsilon$ to denote an empty string. So a language $L$ over the alphabet $\Sigma$ is a collection of strings.
